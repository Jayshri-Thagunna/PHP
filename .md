<h1>Functions</h1>
1) strlen() - Length of a string. (all whitespace count)<br>
2) str_word_count() - Number of words in string<br>
3) strrev() - Reverses a string<br>
4) ucfirst() - Only first char into upeercase<br>
5) lcfirst() - only first char into lowercase<br>
6) strpos() - first char position<br>
7) str_replace() - replaces char(old, new)<br>
8) trim() - remove whitespace or strings from both sides<br>
9) strtolower() - string into lower<br>
10) strtoupper() - string to upper<br>
11) strcmp() - compare two string in 0 and 1 (if equal = 0 and equal != 1)
